1、竞买人邮箱收不到用户名、密码

(1)提交“参拍申请”后，修改了邮箱地址，这种情况，需要企业先删除竞买人申请参拍的信息，然后竞买人重新提交一次参拍申请，即可。
(2)QQ邮箱会把这种用户名密码提示邮件归入垃圾邮件中。



2、竞买人无法登录拍卖大厅
（1）错误登录其他拍卖大厅，因此无法进入。

（2）登录拍卖大厅，要使用企业分配的用户名和密码。



3、竞买人无法正常出价

(1)拍卖会没有开始或者暂停的情况下，无法出价。
(2)竞买人未登录竞买大厅，所以无法出价。



4、其他注意事项

1、同一个身份证号码只能注册一次，不可以反复注册
2、用户名和密码忘记，需要发身份证电子版邮件给网络部工作人员找回
3、注册时，需要准确填写手机号码+电子邮件地址
4、申请参拍之后，不要更换联系方式，否则收不到用户名密码，以及验证码的短信，如果必须修改，需要联系企业，先删除之前的申请信息，重新提交参拍申请。
